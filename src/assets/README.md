# Assets 目录说明

## 📁 文件组织

### sounds/ - 提醒声音文件
- `water-drop.mp3` - 水滴声
- `gentle-bell.mp3` - 温和铃声
- `chime.mp3` - 钟声

### icons/ - PWA 图标
- `icon-72x72.png`
- `icon-96x96.png`
- `icon-128x128.png`
- `icon-192x192.png`
- `icon-512x512.png`

### animations/ - 动画资源
- `water-wave.json` - Lottie 水波动画（可选）

## 📝 注意事项

1. **声音文件**：建议使用 MP3 格式，文件大小控制在 50KB 以内
2. **图标文件**：必须提供多种尺寸以适配不同设备
3. **动画文件**：Lottie 动画为可选，可提升视觉效果
